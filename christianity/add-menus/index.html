4. Add Menus

Add pages or links to the main, social, docs, or footer menu. Set page level navigation.

Open ./config/_default/menus/menus.en.toml in your code editor.

Global navigation#

Add to main menu#

[[main]]
  name = "Docs"
  url = "/docs/prologue/introduction/"
  weight = 10

[[main]]
  name = "Blog"
  url = "/blog/"
  weight = 20

[[main]]
  name = "Community"
  url = "https://github.com/h-enk/doks/discussions"
  weight = 50

Section navigation#

Add to docs menu#

Add first level menu items (make sure the identifiers are unique):

[[docs]]
  name = "Prologue"
  weight = 10
  identifier = "prologue"
  url = "/docs/prologue/"

[[docs]]
  name = "Recipes"
  weight = 20
  identifier = "recipes"
  url = "/docs/recipes/"

[[docs]]
  name = "Help"
  weight = 50
  identifier = "help"
  url = "/docs/help/"

Add second level menu items in the frontmatter of a documentation page, by referencing a first level menu item (using it’s identifier):

..
menu:
  docs:
    parent: "prologue"
weight: 010
..

Page navigation#

Set levels#

The On this page (Table of Contents) section is automatically generated. In ./config/_default/markup.toml, set the levels you’d like to show (default only <h2> and <h3> headings are shown):

[tableOfContents]
  endLevel = 3
  ordered = false
  startLevel = 2

Footer navigation#

Add to footer menu#

[[footer]]
  name = "Privacy"
  url = "/privacy-policy/"
  weight = 10
